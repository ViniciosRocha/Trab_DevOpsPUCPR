# Trab_DevOpsPUCPR

Repositório destinado à disciplina de DevOps da PUCPR.

## Projeto: Validador de CPF

Implementação em Python do algoritmo oficial de validação de CPF (Cadastro de Pessoas Físicas), incluindo cálculo dos dígitos verificadores.

## Funcionalidades

- Validacao de CPF com/sem formatacao (pontos e traco)
- Algoritmo oficial dos digitos verificadores
- Deteccao de CPFs invalidos (todos digitos iguais, tamanho incorreto)
- Formatacao automatica no padrao XXX.XXX.XXX-XX
- Interface interativa para testar multiplos CPFs
- Testes unitarios com unittest
